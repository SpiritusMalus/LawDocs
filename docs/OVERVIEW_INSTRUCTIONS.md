# Инструкция по PROJECT_TECH_OVERVIEW.md

## Для Claude Code (как обновлять)

После каждого значимого изменения в проекте выполни:
"Обнови docs/PROJECT_TECH_OVERVIEW.md согласно инструкции в docs/OVERVIEW_INSTRUCTIONS.md"

Правила обновления:
- Обновляй ТОЛЬКО затронутые секции
- Не переписывай весь файл заново
- Добавь строку в ## Changelog: [ДАТА] [что изменилось]
- Дата в формате YYYY-MM-DD

Что считается значимым изменением:
- Новая фича или страница
- Изменение флоу пользователя
- Новая таблица или поле в БД
- Новый API эндпоинт
- Изменение архитектурного решения
- Новая зависимость
