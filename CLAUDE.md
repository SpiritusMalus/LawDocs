# LawDocs

B2C сервис: AI-помощник, который превращает житейскую проблему в готовый
юридический документ (претензия в магазин/банк/работодателю/страховой/УК
и т.д.) за 5 минут. Цена — 500 ₽ за документ.

Это **wizard, не чат-бот**. Шаблоны пишет юрист-партнёр, LLM наполняет данными
из формы. Без шаблонов — мусор; без LLM — обычный Doczilla.

## Этап

**Фаза 1 (сейчас): валидация спроса.** Лендинг + форма заявки → Telegram-бот.
Заявки обрабатываются вручную, оплата по QR Тинькофф самозанятого.
Точка отсечения через месяц: 20+ платящих с CAC <500 ₽ → строим MVP.
Иначе — закрываем эксперимент, теряем ~100k ₽ и месяц, а не полгода жизни.

**Фаза 2 (после валидации): MVP**. FastAPI + Postgres + Next.js + GigaChat
+ ЮKassa + S3. 7 типовых шаблонов (магазин, маркетплейс, банк, работодатель,
ОСАГО, УК, авиакомпания). 8 недель part-time.

## Структура репозитория

```
LawDocs/
├── frontend/        # Next.js 16, React 19, Tailwind v4, shadcn, base-ui
└── backend/         # (будет создан в Фазе 2) FastAPI + Postgres
```

Frontend — монолит, который сейчас обслуживает только лендинг. После валидации
в нём же будет wizard + ЛК. Backend появится, когда понадобится оплата
и генерация документов.

## Стек, опинионированно

- **Frontend**: Next.js 16.2 (App Router), React 19, Tailwind v4, base-ui/react,
  shadcn-стиль `base-nova`, lucide иконки, Inter шрифт.
- **Backend** (Фаза 2): FastAPI + Pydantic + SQLAlchemy 2.0 + Alembic + httpx
  + python-docx + weasyprint. Postgres 16, без Redis/Celery/RabbitMQ на старте.
- **LLM**: GigaChat по умолчанию (дешевле, в РФ, 152-ФЗ). Claude через прокси
  как fallback, если качество не вытянет.
- **Платежи**: ЮKassa (после регистрации ИП на УСН 6%). На фазе 1 —
  ручной счёт через Тинькофф самозанятого.
- **Auth** (Фаза 2): magic link по email, JWT в httpOnly cookie. Без паролей.
- **Хостинг**: Selectel/Timeweb (бэкенд + S3), Vercel (фронт), Cloudflare (CDN/DNS).

## Что НЕ делать

- Маркетплейс юристов (исходная идея, отложена в lawair).
- Микросервисы, RabbitMQ, Celery, Redis, Elasticsearch, pgvector/RAG,
  Vault, 2FA, OAuth2 — overkill для фазы 1–2.
- Чат-бот в свободной форме вместо wizard'а.
- Подписку до того, как валидирована разовая модель.
- Бесплатные документы как приманку — обесценивает.

архитектура, шаблоны) — детали, которые ничего не значат, если люди не платят.
